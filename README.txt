Expense Tracker Basic - Source Code

To build the APK:
1. Download and install Android Studio from: https://developer.android.com/studio
2. Open this project in Android Studio.
3. Let it sync Gradle (it may take a minute).
4. Click Build > Build Bundle(s) / APK(s) > Build APK.
5. Once built, click 'locate' to find the APK file on your system.

This project includes:
- Expense & Income tracking
- Category management
- Export features (CSV, Excel, PDF)
- Light/Dark mode
- Hindi/Marathi/English support
